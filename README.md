# iowaliquordb

Relational database for Iowa Liquor Sales



This project is currently under construction



This is a relational database that takes the flat schema from the website and turns it into a star schema. Some of the features included is fixing and accounting for the misspellings or errors related to data entry including things like extra spaces between words and apostrophe errors at the end of brand names. With this database you will be able to perform analysis on businesses and their individual franchises (if they have them), as well the brands and their flavors or promos. Currently this database is version 1.0 and currently is not fulfilling the full dataset as the edge cases are still being worked on and will be corrected soon, this is for the business names, franchise names, and brand names.
